# Iventario De Productos

Este proyecto es una pagina para el inventario y gestion de todo tipo de productos,
realiando las compras y ventas y manejando roles especificos para cada uno

## Tecnologias

Este proyecto fue realizado con PHP puro, aplicando estilos con Boostrap y JS y manejando
variables de sesion estrictas
