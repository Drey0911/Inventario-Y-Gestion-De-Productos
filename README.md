# Iventario De Productos

Este proyecto es una pagina para el inventario y gestion de todo tipo de productos,
realiando las compras y ventas y manejando roles especificos para cada uno y sesiones
con PHP, practica para CRUD Web

## Tecnologias

Este proyecto fue realizado con PHP puro, aplicando estilos con CSS y Tailwind,
alertas y parte de la funcionalidad con JS y una conexion a BD MySql
